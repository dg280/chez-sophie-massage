⚙️ Configuration GitHub

Connectez votre depot GitHub pour publier les modifications des soins et tarifs.

🔗 Connexion au depot

Entrez vos identifiants GitHub. Ils sont chiffres et stockes sur cet appareil.

🔒 Le token est chiffre avec votre mot de passe admin (AES-GCM) avant d'etre stocke. Le repo doit contenir le fichier tarifs.json. Creer un token avec permission Contents: Read & Write uniquement.

🔒 Securite

🔑 Changer le mot de passe

Le mot de passe est stocke sous forme de hash securise (SHA-256). Minimum 8 caracteres.
